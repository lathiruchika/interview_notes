# 🚀 sorted() Function in Python (Interview Notes)

## 🔹 Definition
sorted() returns a new sorted list from an iterable.

## 🔹 Syntax
sorted(iterable, key=None, reverse=False)

## 🔹 Example
nums = [3,1,2]
print(sorted(nums))

## 🔹 Use Cases
- Sort numbers
- Sort strings
- Custom sorting using key
- Descending order using reverse=True

## 🔹 Rules
1. Returns new list
2. Works on any iterable
3. Original data unchanged
4. key must be a function
5. reverse=True for descending

## 🔹 Interview Tip
sorted() supports custom sorting using key parameter.
