# 🚀 Lambda Function in Python (Interview Notes)

## 🔹 Definition
A lambda function is a small anonymous function defined using the `lambda` keyword.
It can take any number of arguments but only one expression.

## 🔹 Syntax
lambda arguments: expression

## 🔹 Example
add = lambda a, b: a + b
print(add(2, 3))  # 5

## 🔹 Rules
1. Only one expression allowed
2. No statements (if/loops/return not allowed)
3. No assignment
4. Multiple arguments allowed
5. No return keyword

## 🔹 Use Cases
- map()
- filter()
- sorted()

## 🔹 Interview Tip
Lambda is an anonymous function used for short, single-expression operations.
